## API Testing — Интернет-магазин  

### Объект тестирования  
REST API [интернет-магазина](https://intern.demoshopping.ru/)  

---

## Цель тестирования  
Проверка корректности работы API для:
- получения списка товаров  
- получения информации о товаре
- фильтрация товаров 
- добавления товара в корзину  

---

## Инструменты  
- Postman  
- Swagger

---

## Что тестировалось  

### Товары (Products API) 

Проверки:
- корректность статус-кода (200 OK)  
- наличие обязательных полей (id, name, price)  
- корректность данных товара  

---

### Корзина (Cart API) 

Проверки:
- товар корректно добавляется в корзину  
- обновляется количество товаров  
- корректный расчёт суммы  

---

## Результаты тестирования  
- Проверено: базовые API сценарии  
- Критических ошибок не выявлено
